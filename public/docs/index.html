Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Create User

Example request:
$client = new \GuzzleHttp\Client();
$url = 'http://localhost/api/register';
$response = $client->post(
    $url,
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'json' => [
            'name' => 'svtwewfvge',
            'email' => 'karianne44@example.org',
            'password' => '~T-3:bNGWtWqG%47-|B',
            'password_confirmation' => 'sed',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
const url = new URL(
    "http://localhost/api/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "svtwewfvge",
    "email": "karianne44@example.org",
    "password": "~T-3:bNGWtWqG%47-|B",
    "password_confirmation": "sed"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

POST api/register

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

Body Parameters

name   string   

Must not be greater than 255 characters. Example: svtwewfvge

email   string   

Must be a valid email address. Example: karianne44@example.org

password   string   

Must be at least 6 characters. Example: ~T-3:bNGWtWqG%47-|B

password_confirmation   string   

Example: sed
